AGRO-MAR v24 — galeria i panel admina

Zmiany:
- poprawiony wygląd galerii,
- znikają brzydkie szare pola po bokach zdjęć,
- zdjęcia są większe i bardziej eleganckie,
- kafelki mają ciemne tło i efekt premium,
- podpisy zdjęć są mniej nachalne,
- przyciski w panelu admina są dużo bardziej widoczne:
  Zapisz — ciemnozielony
  Usuń — czerwony
  Dodaj — złoty/ciemny

Instrukcja:
1. Nie usuwaj ręcznie danych w Supabase.
2. W GitHub nadpisz obecne pliki zawartością v24.
3. Commit changes.
4. Poczekaj na deploy Vercel.
5. Odśwież stronę CTRL+F5.
6. Sprawdź galerię i panel /admin.

Nie trzeba uruchamiać nowego SQL, jeśli v23 był już uruchomiony.
